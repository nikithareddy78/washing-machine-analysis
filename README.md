# Washing Machine Sensor Data Analysis

## Project Overview
This project analyzes washing machine sensor data using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed
- Data Loading
- Data Cleaning
- Missing Value Analysis
- Correlation Heatmap
- Temperature Analysis
- Power Analysis
- Energy Consumption Analysis

- Maximum Power: 2337.55
- Total Energy: 85473869.0
